A union is a special data type available in C that allows 
storing different data types in the same memory location. You can define a union with many members,
 but only one member can contain a value at any given time. Unions provide an
 efficient way of using the same memory location for multiple purposes. 

A structure is a user-defined data type available in C that allows to combining data items of different kinds.
 Structures are used to represent a record.
